# QuizApplication
A minimalist, console-based Java quiz application with hardcoded questions.

## Prerequisites

- Java Development Kit (JDK) 8 or higher.

## Getting Started

- Clone: 
```bash
git clone https://github.com/Azok42/QuizApplication.git
```

- Compile: 
```bash
cd QuizApplication
javac $(find . -name "*.java") -d bin
```

- Run:
```bash
cd bin

java quiz_application/Quiz
# OR
java quiz_application/Quiz print # for print mode 
```
